StopYappingBro

Privacy Policy — Last updated 2026-06-11

StopYappingBro does not collect, transmit, or store any user data beyond what you explicitly configure. Your custom summarization prompt is saved in chrome.storage.local on your own device and is never sent off-device by this extension.

What the extension reads

On a YouTube video page, the extension reads the video's transcript. It does this by injecting a small script into the page's own JavaScript environment that observes YouTube's internal transcript API responses (the get_panel and get_transcript requests YouTube makes for its own transcript panel). The transcript text is parsed locally inside your browser.

Where the transcript goes

  • Copy Transcript writes the transcript to your system clipboard. Nothing leaves your device.
  • Summarize opens a new tab to chatgpt.com and fills the message composer with your prompt followed by the transcript, then submits it. The transcript is sent only to ChatGPT, in your own logged-in browser session, exactly as if you had pasted and sent it yourself. ChatGPT's handling of that message is governed by OpenAI's own privacy policy.

What the extension does NOT do

  • Does not make any network requests of its own.
  • Does not send data to the developer or any third-party analytics service.
  • Does not read cookies, passwords, or form fields.
  • Does not track your browsing history or your activity across sites.
  • Does not run any remote code — all logic ships inside the extension package.

Storage

The extension stores a single value — your custom summarization prompt — in chrome.storage.local. This storage is sandboxed per-extension and is not shared with websites or other extensions. If you never change the prompt, nothing is stored at all and the built-in default is used.

Permissions

The extension declares the following permissions. See the project README for the full per-permission rationale provided to the Chrome Web Store.

  • storage — save your custom summarization prompt across browser sessions.
  • Host access to https://www.youtube.com/* — inject the transcript reader and the on-page buttons.
  • Host access to https://chatgpt.com/* — fill the ChatGPT composer with the transcript and submit it.

Contact

For questions about this policy, please open an issue on the project's GitHub repository.

StopYappingBro is an open-source project. No data leaves your browser except the transcript you choose to send to ChatGPT.
